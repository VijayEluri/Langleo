Langleo

Hi! Welcome to Langleo :) Langleo lets you learn vocabulary really efficiently by trying to predict when you'll forget the given word and asking you about just before it happens. The introductory section of this help page explains basic ideas behind Langleo and tells you how to use it efficiently. Other sections describe all screens in the application. Remember that at any time when using Langleo you can press the 'Menu' button and then 'Help' to get information about the screen you're currently in.
