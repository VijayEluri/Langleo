Langleo

Hi! Welcome to Langleo :) Langleo lets you learn vocabulary really efficiently by trying to predict when you'll forget the given word and asking you about just before it happens. First few introductory sections of this help page explain basic ideas behind Langleo and tell you how to use it efficiently. Other sections describe all the screens in this application.

Remember that at any time when using Langleo you can press the 'Menu' button and then 'Help' to get information about the screen you're currently in.

Introduction

Langleo uses an approach similar to the Supermemo method. It means it will try to predict when you'll forget a word and will ask you about it just before it happens.

Langleo's main goals are effectiveness and efficiency. I've personally learned 1200 new spanish words during the first 30 days of using it (40 words per day on average) and I believe it's possible to learn 80 words per day. That means you could learn 5000 words (usually enough to be able to communicate in a given language freely) in 2-3 months. I don't think any other tool available for the Android platform can help you achieve such results.

A certain number of new words is introduced daily (can be changed in preferences). Langleo will ask you about them later depending on your general performance and calculated difficulty of the word.

Words are organized into lists and lists into collections. So for example you can have a collection named 'French' and inside of it few lists like 'Weather', 'Colors', 'Body parts' etc. Collections hold informations like base and target languages, collection's priority and name.

Studying

When you're studying a word will be show to you. Read it and decide whether you know it or not. Click somewhere on the screen to see the correct answer. If you remembered it correctly then click 'Correct', otherwise click 'Incorrect'.

When Langleo introduces a new word it shows a 'New word' label on top of the study screen. Try to memorize it and click 'Continue'. If you knew the word previously then click 'I know it already'. This will indicate that you knew this word already and Langleo will show one more new word instead if this one.

Using Langleo efficiently

First, absolutely necessary thing for efficient learning is your decision to improve your level of foreign language skills. Learning vocabulary is a long process, which requires quite a lot of commitment and motivation. So before actually starting to learn make sure that you're motivated enough to keep learning at least for few weeks. If you'll get bored after few days then the entire process will turn out to be just a waste of your time.

Langleo should be used at least once, and preferably 2-3 times a day. Always answer all the questions in the current session. If you're constantly unable to do so then try decreasing the number of new words learned daily in the preferences window. If you think you could learn more words today then you can always add more new words for this single day in the Study window (Menu->More new words).

Tips

  • Make sure all the words you learn are fully understandable. If you have problems remembering a certain word then it may be a good idea to reformulate it
  • When studying make sure that after seeing the correct answer you spend few seconds repeating it in your head. Seeing the answer alone has little or no effect.
  • For every word try to write (or find) a sentence which uses it and put it in the word's 'Note' field. This sentence will then be shown during studying. It tremendously improves learning time and shows you a real life example.

Importing and exporting words

Langleo lets you import and export words from/to text files. This is especially useful when you want to learn words obtained from other source, for example a web site or you just prefer to create lists of words on your computer instead of your phone. Each text file will be imported as a separate list with the same name as the file.

Format of text files

Imported text files should contain one word per line with the original word, its translation and note associated with it (in that order) separated by one character (or string), preferably a tab. Notes are not mandatory, so it's perfectly OK if a line contains just the word and its translation. Lines with a format different that that will be ignored.

Importing files

Let's say that you have created few text files on your computer and you'd like to import them to Langleo. First you have to copy them on your SD card into the Langleo directory (you can of course create a deeper directory structure in there). Then in the app go to 'Collections' and there select or create a collection to which you'd like to import the files. Click on the collection and then press 'Menu->Import'. On the following screen select the files and click 'OK'. Later you can change the import options if needed - make sure everything under 'Will be imported as' looks right. After you click 'Import' the import process starts. Langleo is able to import approximately 20 words per second so it can take some time depending on the size of your files, although it shouldn't be an issue at all because the entire process takes place in the background.

Exporting files

If you want to export words from your collection then click 'Collections'. Then click on the selected collection and press 'Menu->Export'. On the following screen you can select lists which will be exported. After you click 'Export' the files should appear on your SD card in the Langleo/<name of collection> directory.

Study screen

This screen is definitely the most important one in the app. Here the learning process takes place :)

In the upper part of the screen you can see flags of both languages (target language on top, base language under it). To the right of these there's a progress bar representing the progress of the current session, with the number of questions answered correctly and number of total questions in this session under it. You can also see the elapsed and estimated total time of the session.

In the upper right corner is located the audio button. If activated the application will read both the question and answer at the appropriate time. When this button is activated you can also use gestures to indicate whether you knew the answer or not instead of the buttons. Moving your finger up on the touchscreen will have the same effect as the 'Correct' button. Moving it down will have the same effect as the 'Incorrect' button. This combination of audio and gestures lets you learn new words without even looking at the screen! It means you can do it with headphones on and your phone in your pocket. Read more about Langleo's Text-To-Speech capabilities in the section below.

On the bottom of the screen there are always two buttons. When new word is introduced they are 'I know it already' and 'Continue'. The second one shows the next word. The first one also shows the next word, but doesn't count the current word as new, so Langleo will actually show one more new word. If the shown word is not new then the two buttons are 'Incorrect' and 'Correct'. Use them to indicate whether you knew the answer or not.

If you feel like learning more new words today than the current limit then you can increase it in 'Menu->More new words'.

You can also quickly delete or edit the word you're seeing now by accessing these options after pressing 'Menu' button.

Text-To-Speech

Please note that Langleo currently uses Text-To-Speech library to convert text to audio. Its current version is far from perfect, but the quality is going to improve in the next version of Android. It also doesn't read properly certain characters, especially the ones with accents above them.

Collections screen

This screen lists all the collections you currently have. Each item in the list contains it's name, flags of languages, overall statistics and progress information. If a collection is disabled only it's name and flags of used languages are shown.

To see contents of a collection simply click on it. To create a new collection press 'Menu->New'. Click and hold on a collection to edit or delete it.

Edit collection screen

This screen lets you edit the selected collection. You can change it's name, base and target languages and priority. Priority controls the number of new words that will be taken from this collection, higher means more new words will come from this collection.

You can also disable this collection. This means no new words will come from this collection.

Lists screen

This screen shows all lists in a given collection. You can add a new list by pressing 'Menu->New list'. You can also download lists from internet by clicking 'Menu->Download' or import them from text files ('Menu->Import'). Exporting lets you save the words from selected lists in text files on your SD card.

Edit list screen

This just lets you change the name of the list.

Words screen

Lists words in current list, the first line contains the word and the second its translation. Click on the word to edit it.

Edit word screen

Here you can change the word, its translation and the note attached to it (note is shown under the translation when studying). Clicking 'New' saves current word and creates a new one.

File selection screen

This screen lets you select files which you want to import. Remember that each file will be loaded as a separate list and all selected files must use the same character as field separator.

Import screen

On this screen you can change options necessary for proper importing of the selected files (read Importing and exporting files section first). On the bottom of the screen you can see an example of a line taken from the selected files. Under it you can see how it will be interpreted. Change field separator and switch word/translation order if necessary.

List selection screen

Here you can select lists from your collection which you'd like to export. Selected lists will be saved on your SD card in Langleo/<name of collection> directory.

Download screen

This screen shows the available lists which can be downloaded for target language of current collection. The lists are downloaded directly from studystack.com so active internet connection is required. Click on the list you'd like to download to your collection. Use the search box on the top for quicker navigation.

Langleo updates this list every few days, if for some reason you want to force an update then use 'Menu->Refresh'.

List details screen

Here you can see the contents if the selected list before downloading it. Make sure that labels above columns match the languages in the actual content. If they don't then use the 'Switch word/translation order' box.
