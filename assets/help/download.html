Download screen

This screen shows the available lists which can be downloaded for target language of current collection. The lists are downloaded directly from studystack.com so active internet connection is required. Click on the list you'd like to download to your collection. Use the search box on the top for quicker navigation.

Langleo updates this list every few days, if for some reason you want to force an update then use 'Menu->Refresh'.
